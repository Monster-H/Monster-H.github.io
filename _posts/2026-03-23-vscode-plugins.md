推荐一些我日常使用的 VS Code 插件，大幅提升开发效率！🚀

## 🏆 必装插件

### 1. GitLens — Git 增强
> 在编辑器中查看 Git 历史、提交记录、代码作者

```
extensionId: eamodio.gitlens
```

### 2. Prettier — 代码格式化
> 自动格式化代码，保持风格统一

```
extensionId: esbenp.prettier-vscode
```

### 3. ESLint — 代码检查
> JavaScript/TypeScript 代码质量检查

```
extensionId: dbaeumer.vscode-eslint
```

## ⚡ 快捷键技巧

```
Ctrl + Shift + P  # 命令面板
Ctrl + D          # 选择下一个相同词
Alt + Click       # 多行编辑
Ctrl + Shift + L  # 选择所有相同词
```

---

## 📥 安装方式

1. 打开 VS Code Extensions 面板 (`Ctrl + Shift + X`)
2. 搜索插件名称
3. 点击 Install

或者直接复制上面的 `extensionId` 在Extensions面板搜索安装！
