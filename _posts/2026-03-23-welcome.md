欢迎来到我的技术博客！🎉

## 📖 关于这个博客

这是一个使用 **Minimal Mistakes** Jekyll 主题搭建的博客，托管在 GitHub Pages 上。

在这里你会找到：

- 💻 **技术教程** - 从入门到进阶的学习笔记
- 🛠️ **项目实践** - 实际开发中的经验总结
- 📝 **工具分享** - 好用的软件和插件推荐
- 💡 **思考感悟** - 技术成长的心路历程

## 🚀 技术栈

本博客使用以下技术：

| 技术 | 说明 |
|------|------|
| Jekyll | 静态网站生成器 |
| Minimal Mistakes | GitHub Pages 热门主题 |
| GitHub Pages | 免费托管服务 |
| Markdown | 内容编写格式 |

## ✍️ 如何写博客

博客文章存放在 `_posts` 目录下，文件名格式为：

```
年-月-日-文章标题.md
```

文章使用 Markdown 编写，支持代码高深、公式渲染等功能。
